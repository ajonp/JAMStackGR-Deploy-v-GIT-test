# How to use this library in Angular
`npm i jamstackgr2-library`

## Include in module
app.module.ts
```
import { Jamstackgr2LibraryModule } from 'jamstackgr2-library';
...
  imports: [
    Jamstackgr2LibraryModule
  ],
...
```

## Use in component
app.component.html
```
<lib-jamstackgr2-library></lib-jamstackgr2-library>
```
